# PetAdoption

Technologies used are JavaScript, ReactJS, CSS, HTML
●  	Designed a React based Single Page Application using ReactJS, CSS to help people adopt pets which are in need of a home		
●  	Implemented React hooks such as UseState, UseEffect and custom routing with a proper demonstration of functionality such as multiple pages for the Single Page Application, navigation that allows the user to move among the pages, validations, hamburger menu, gallery of images, modal dialogue, etc	
●  	Used CSS to make the web page responsive/adaptive based on the screen size and transitions on the menu dropdown
